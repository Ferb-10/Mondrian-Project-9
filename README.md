# Mondrian-Project-9

## Overview
This project recreates a Piet Mondrian–style abstract artwork using only HTML and CSS.
It is built entirely with CSS Grid, focusing on precise layout control, spacing, and color placement.
The main purpose of this project is to deepen understanding of CSS Grid and complex layout design.

![image alt](https://github.com/Ferb-10/Mondrian-Project-9/blob/19dfd370979001c03707459bcdeb82754e0b73cf/Mondrian.png)

## Features
- Pure HTML and CSS implementation
- Layout created entirely with CSS Grid
- Precise control using grid-template-rows, grid-template-columns, and grid-area
- Accurate color blocks inspired by Mondrian’s artwork
- No images or external libraries required

## Technologies Used
- HTML5
- CSS
- CSS Grid

## What I Learned
- Designing complex layouts using CSS Grid
- Controlling element placement with grid-area
- Managing gaps and borders for visual separation
- Recreating artistic designs with code
- Understanding grid-based layout systems used in modern web design
